# K-Means Clustering Project (Customer Segmentation)

## 📌 Description
Group customers based on:
- Annual Income
- Spending Score

## 📂 Files
- main.py → clustering script
- data.csv → dataset (download from Kaggle)
- clustered_data.csv → output

## 🚀 Steps

1. Install:
pip install pandas scikit-learn matplotlib

2. Download dataset:
https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

Rename Mall_Customers.csv → data.csv

3. Run:
python main.py

## ⚠️ Required columns:
Annual Income (k$)
Spending Score (1-100)
